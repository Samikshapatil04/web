HTML Notes

<h1> to <h6> Header Tags

HTML provides several header tags, ranging from <h1> to <h6>, which represent different levels of headings.

<strong> Bold

The <strong> tag is used to make the text bold. It emphasizes the importance of the text.

<em> Italic

The <em> tag is used to make the text italic. It is used to emphasize or stress the text.

<u> Underline

The <u> tag is used to underline the text. It is used to draw attention or indicate a hyperlink.
